WHAT I'VE BEEN WORKING ON

RESTful CRUD API on .NET Core with MongoDB

  • This API was written in C# and uses asynchronous programming to increase speed and efficiency.
  • A MongoDB environment is created using Docker and CRUD operations can be implemented through the Postman app.
  • By taking advantage of DTO’s this API is ready to adapt as needs change and ensures decoupling. Code is even more loosely coupled from IoC and dependency injections.
  • Using an open-source program, Health Checks are built in to perform status requests.
GitHub Repository

ASP .NET Core MVC BestBuy Dynamic Website

  • A fully dynamic website written in C#, HTML, CSS, and JavaScript. This website uses MySQL database service on the backend to create a mock BestBuy database of products.
  • CRUD operations allow users to add new products (Create), pull existing products (Read), update existing products (Update), and remove products from the database (Delete).
GitHub Repository

GeoLocation Application for Taco Bell Locations

  • This application parses a CSV file containing information on hundreds of Taco Bell locations. Each location is separated out and given values of its location’s name and longitude and latitude coordinates.
  • By using the GeoCoordinatePortable class, the application calculates the distance between all locations on the CSV file and returns the two that are farthest apart.
  • Written in C# on .NET Core.
GitHub Repository

Busy Bear Education Website Design

  • The website for this company was designed so students, parents, or teachers can easily find the information they are looking for.
  • The layout of the pages, routes of links, and regular maintenance and updates are all done by me.
  • The website was created using Squarespace and then built upon. A majority of the images were created on Canva to fit the needs of the page.
